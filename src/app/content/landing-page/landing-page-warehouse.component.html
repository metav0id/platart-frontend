User guide: {{role}}

Things you can do in the {{warehouse}} module:
- Check how much stock of each item is in warehouse
Warehouse Stock in warehouse

-En stock in warehouse se observa una lista de todo el stock con la información correspondiente.

Warehouse Stock in warehouse

-La información se puede filtrar utilizando el filtro sobre la tabla

- Register a new delivery to a shop
Warehouse New delivery to shop

-Inserte información sobre el envío que se hará a alguna
-Consulte el número de objetos disponibles en stock.

Warehouse New delivery to shop

-Introduza información sobre un descuento, si aplicable.

Warehouse New delivery to shop

-Con el envío una vez en la lista, se puede guardar localmente, borrar o guardar/enviar con los botones de arriba. No olvide seleccionarlo a la izquierda.

- Register an incomming delivery to warehouse
Warehouse New delivery to warehouse

-Llene el formulario con información sobre una nueva entrada de mercancía de un distribuidor al almacen.

Warehouse New delivery to warehouse

-Con el envío una vez en la lista, se puede borrar o guardar/enviar con los botones de arriba. No olvide seleccionarlo a la izquierda.

Tip: Select one on the list above or use the menu at the header to navigate across the webpage!

Things you can do in the {{manager}} module:
- Register new warehouses and shops, and locate them in the map.
Manager Map

-Registra nuevos almacenes y/o sucursales. Presiona "Create new comerce" e introduce la informacion del comercio en los campos correspondientes. Despues presiona "save" para guardar.

Manager Map

-Hacer click en el mapa en ubicacion elegida, y mediante la ventana emergente que seleccionar el comercio que se quiere registrar y hacer click en aceptar.

Manager Map

-Una vez con el comercio registrado, se puede hacer click en el marcador y mediante los botones se puede editar la informacion, borrar el marcador o borrar el comercio de la base de datos.

- Manage and consult shop´s and warehouse´s information.
Manager Manager shop information

-Utiliza los filtros para elegir la informacion que se quiere ver en la tabla.

- Cunsult business inteligence.
Manager Business inteligence

-Revisa toda la infomacion estatistica sobre las ventas.

- Consulta el desempeño en ventas los objetos vendidos.
Manager Category overview

-Consulta el desempeño de las categorias en un marco de tiempo.

- Consulta el desempeno de las tiendas.
Manager Shops ranking

-Consulta diariamente el desempeño de las tiendas individualmente.

- Register a new user for the platform.
Manager Register new user

-Registra un nuevo usuario para la plataforma introducionedo la informacion necesaria en el formulario. Recuerda que las tiendas que eligas seran a las que el usuario tendra acceso para administrar.

- Use tool to consult information about checkin items in warehouse by using a filter.
Manager Warehouse checkin

-Utilize los calendarios para construir un filtro que consulte los objetos en stock dentro de cierto marco de tiempo.

- Add a new category of an item to the warehouse stock registry
Warehouse New category

-Utilice el "checar disponibilidad" para verificar si la categoria ya existe o esta disponible. -Si la categoria esta disponible, puede agregarla a la lista con el boton de crear categoria nueva.

Warehouse New category

-Con la categoria una vez en la lista, se puede borrar. No olvide seleccionarlo a la izquierda.

Tip: Select one on the list above or use the menu at the header to navigate across the webpage!
